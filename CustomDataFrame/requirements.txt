# Custom DataFrame Implementation - Requirements

## System Requirements
- .NET 8.0 SDK or later
- Windows, macOS, or Linux

## Installation Commands
```bash
# Navigate to folder and run
cd CustomDataFrame
dotnet restore
dotnet build
dotnet run

# Or run with specific Excel file
dotnet run path/to/your/file.xlsx
```

## Key Features
- Complete Excel workflow (read → analyze → export)
- Multi-format export: Excel, CSV, PDF
- Pandas-like DataFrame operations
- Multi-worksheet support
- Statistical analysis
- Data filtering and sorting

## Best For
- Excel processing workflows
- Small to medium datasets (<10,000 rows)
- Complete control over implementation
- Educational purposes
